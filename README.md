# 资源下载中心

一个简洁的资源下载聚合页面，提供比特币相关的电子书、视频教程、文档资料等资源的下载链接。

## 功能特点

- 📚 电子书资源下载
- 🎥 视频教程分享
- 📄 文档资料整理
- 📱 移动端适配
- 🔍 SEO 优化

## 技术栈

- HTML5
- CSS3 (Grid Layout, Flexbox)
- JavaScript (原生)

## 本地预览

使用 HTTP 服务器运行项目：

```bash
# 使用 Node.js http-server
npm install -g http-server
http-server
